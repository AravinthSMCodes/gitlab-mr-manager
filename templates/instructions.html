How to Use Searchable Dropdowns

Step-by-Step Guide

Step 1: Click on a Filter Dropdown

Click on any of the filter dropdowns (Author, Label, or Reviewer) in the filters section.

Step 2: Type to Search

Once the dropdown opens, you'll see a search box. Type the name you're looking for.

Step 3: Select an Option

Click on the option you want to filter by, or use arrow keys and press Enter.

Step 4: View Filtered Results

The MR list will automatically update to show only MRs matching your filter.

Keyboard Shortcuts

  • Arrow Down/Up: Navigate through options
  • Enter: Select highlighted option
  • Escape: Close dropdown
  • Type: Filter options by typing

Multiple Filters

You can combine multiple filters:

  • Filter by Author AND Label
  • Filter by Reviewer AND Author
  • Filter by all three (Author, Label, Reviewer)

All filters work with AND logic - MRs must match ALL selected filters.

Clear Filters

To clear all filters and see all MRs again, click the "Clear Filters" button.

Troubleshooting

If dropdowns don't open:

  • Make sure JavaScript is enabled in your browser
  • Try refreshing the page
  • Check browser console for errors (F12 → Console)

If filtering doesn't work:

  • Check that the URL updates when you select a filter
  • Verify the filter parameters are in the URL (e.g., ?author=username)
  • Try using the "Clear Filters" button and selecting again
